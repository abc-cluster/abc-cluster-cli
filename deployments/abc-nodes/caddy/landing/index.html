Install abc CLI

Use the official installer to fetch the correct binary for your platform (Linux / macOS / Windows).

System-wide install (/usr/local/bin/abc)

curl -fsSL -H "Accept: application/vnd.github.raw+json" \
  "https://api.github.com/repos/abc-cluster/abc-cluster-cli/contents/scripts/install-abc.sh?ref=main" \
  | sh -s -- --sudo

User-local install (~/bin/abc, no sudo)

curl -fsSL -H "Accept: application/vnd.github.raw+json" \
  "https://api.github.com/repos/abc-cluster/abc-cluster-cli/contents/scripts/install-abc.sh?ref=main" \
  | sh -s --

After install: abc auth login → abc --help

Platform operators

Grafana Dashboards — metrics & logs Prometheus Metrics store & query Loki Log aggregation (API) Grafana Alloy Telemetry collector UI Traefik Load balancer & service catalog Consul Service discovery & health Nomad UI Job scheduler — full view MinIO S3 API S3-compatible endpoint (mc / SDK) RustFS Alternate S3-compatible store Vault Secrets & SSH certificate authority Boundary SSH session broker — access any node
Stack: Tailscale (overlay network) → Caddy (vhost routing / TLS) → Traefik (Consul-catalog load balancer) → Nomad services · Service discovery via Consul · Secrets & SSH CA via Vault · SSH sessions brokered by Boundary

CLI quick-start (abc on PATH)

Nomad and service operations

export ABC_ACTIVE_CONTEXT=abc-bootstrap
abc admin services nomad cli -- job status
abc admin services nomad cli -- job status abc-nodes-grafana
abc admin services nomad cli -- job run -detach deployments/abc-nodes/nomad/uppy.nomad.hcl

Traffic enters via aither.mb.sun.ac.za (LAN) and is proxied by Caddy through Traefik to backend services on private / Tailscale addresses. Service vhosts are at *.aither — add 100.70.185.46 as a split-DNS nameserver for .aither in the Tailscale admin console to resolve them on any tailnet device.
