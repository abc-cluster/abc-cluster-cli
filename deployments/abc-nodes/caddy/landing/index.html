Install abc CLI (script)

Use the official installer script from the repository to fetch the correct binary for your platform.

Install to /usr/local/bin/abc

curl -fsSL -H "Accept: application/vnd.github.raw+json" \
  "https://api.github.com/repos/abc-cluster/abc-cluster-cli/contents/scripts/install-abc.sh?ref=main" | sh -s -- --sudo

After install, run abc auth login and then abc --help.

Advanced CLI use (abc on PATH)

For users who already have abc in PATH and want direct command-line operations:

Nomad and service operations

export ABC_ACTIVE_CONTEXT=abc-bootstrap
abc admin services nomad cli -- status
abc admin services nomad cli -- job status abc-nodes-grafana
abc admin services nomad cli -- run -detach deployments/abc-nodes/nomad/uppy.nomad.hcl

Traffic is proxied by Caddy while backend services remain on private or Tailscale addresses.
